Delete All Wordpress Tags With in A Single Click. It is Mass Delete Tags Plugin.

== Description ==

Official Website: http://www.chunoa.com/

WP Delete Tags is a free wordpress plugin to delete all wordpress tags with in a click. It is a mass delete tags plugin to remove all the Wordpress tags at once.

Note:- All tags will be deleted once you click WP Delete Tags menu item in left navigation in WP dashboard. Tags will not be restored after you delete plugin. To add tags to your Wordpress Posts you can download WP Auto Tagging plugin http://www.chunoa.com/product/wp-auto-tagging-plugin/


== Installation ==

1. Upload WP Delete Tags Plugin to WP-Content -> Plugins or Directly Install It from Plugins -> Add New -> WP Delete Tags
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click WP Delete Tags Menu Item in Wordpress Dashboard and all the tags will be deleted automatically.
